## Tier 3 — 어렵거나 유료 (현재 제외)

- 실시간 호가, 옵션 IV 서피스, 소셜(레딧/X) 감성 점수, sell-side 컨센서스 → 라이선스/유료.
- **한국 외국인 수급·KOSPI/VKOSPI 실시간** → KRX 독점. 한국 심리의 가장 강한 신호인데 무료 접근 불가 = 명시된 blind spot. (대용: KRW 환율, 코스피 yfinance 종가.)

## 설계 메모
- 모든 신규 요인은 `engine/psych.py`의 `fear_greed()`에 `comp.append(...)` 한 줄로 추가 = 자동으로 합성·렌더 반영.
- 요인이 늘수록 단순평균 대신 **가중치**(CNN은 동일가중) 고려. 지금은 동일가중 = 투명·견고.
- 각 요인은 독립 degrade. FRED 호출 누적(현재 ~24/build) 모니터링 — 한도 시 retry/backoff가 흡수.
